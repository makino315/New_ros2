# 東京都の明日の天気を取得

## mypkg
- このリポジトリはROS2のパッケージで授業で作成した。

## ノードの説明
- このパッケージは明日の東京の天気をパブリッシュする'weather_pub'というノードで構成されている。
- topicの名前は'weather'である。

## 使い方
以下のコマンドで実行する。
```bash
ros2 run mypkg weather_pub
```
実行しても何も表示されなく、もう一つの端末で次のコマンドを入力し実行する。
 ```bash
 ros2 topic echo /weather
 ```
